# 図書館セルフ貸出システム 機能要件

以下はヒアリング内容をもとに整理した機能要件です。

| 機能ID | 機能名           | 概要                                                                 |
|:------:|:----------------|:---------------------------------------------------------------------|
| F01    | 書籍ID管理      | 書籍IDをCSVで管理し、書籍の概要や貸出状態（貸出可/不可/貸出中/延滞）を表示 |
| F02    | 利用者ID管理    | 利用者ID（氏名、電話、メール、住所等）の発行・追加・削除・管理             |
| F03    | 貸出処理        | 利用者IDと書籍IDを用いて貸出を行い、支払い処理も実施                       |
| F04    | 返却処理        | 営業時間内での返却受付、延滞時の支払い、破損時の人間対応                    |
| F05    | 貸出履歴保存    | 貸出・返却の履歴をCSVファイルに記録                                      |
| F06    | 貸出履歴閲覧    | 利用者ごとの貸出・返却履歴を表示                                         |
| F07    | 延滞者管理      | 延滞者の抽出・管理、延滞料金の計算                                       |
!-- | F08    | 操作ログ管理    | 操作ログをCSV形式で保存                                                 | -->
